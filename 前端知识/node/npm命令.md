# npm

```log
npm install 本身就有一个别名 npm i

# 写入dependencies，发布到开发环境
npm i module_name -S
npm install module_name --save

# 写入devDependencies，发布到生产环境
npm i module_name -D
npm install module_name --save-dev  

# global全局安装(命令行使用)
npm i module_name -g

# 本地安装(将安装包放在 ./node_modules 下)
npm i module_name
```
