# mysql安装

账号：root/Longlong

mysql在windows上有两种安装方式：安装引导程序安装和压缩包方式安装。

注意：

使用引导程序安装的话，选择自定义安装，选择要安装的组件，点击修改安装路径。
使用压缩包方式安装，注意my.ini文件的配置。
可以先使用引导程序安装，之后使用压缩包的方式进行配置。
