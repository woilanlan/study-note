目录组织和约定

- 一类：某类知识的总称（书架）
  - 二类：某本书籍或某套视频相关（书）
    - 三类：章节或主题
      - img
      - 思维导图
      - 文章（md）
      - 资源（Word,Excel,pdf）
      - code
  - 二类：技术面（引导和增强）
    - 入门指南
    - 知识辨析、源码分析
    - 问题汇总
    - 拓展实践

详解

- 一本书或一套视频一定是要作为一个二类的，注重知识体系的连贯性。
- 技术面可以作为某本书的父级（评估）
- 技术面包括入门指南，问题汇总、知识辨析，源码分析，拓展实践。
