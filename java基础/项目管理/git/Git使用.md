# Git使用

配置用户和电子邮件

```bash
$ git config --global user.name "woilanlan"

$ git config --global user.name
woilanlan

$ git config --global user.email "woilanlan@gmail.com"

$ git config --global user.email
woilanlan@gmail.com

$ git help config
```

克隆现有的仓库：
